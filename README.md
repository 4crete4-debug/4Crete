# 4Crete Summer League — Πλατφόρμα

Live πλατφόρμα τουρνουά μπάσκετ με βαθμολογία, bracket, ζωντανό σκορ σε 2 γήπεδα, MVP, στατιστικά και προβολή TV. Συγχρονίζεται real-time μέσω Firebase σε όλες τις συσκευές.

---

## 🚀 Πώς να το ανεβάσεις online (βήμα-βήμα)

### Βήμα 1 — Κλείδωσε τη βάση δεδομένων (κάντο ΤΩΡΑ, μία φορά)

Στο Firebase Console:
1. Πήγαινε **Build → Firestore Database → Rules** (καρτέλα πάνω)
2. Σβήσε ό,τι υπάρχει και κάνε paste το περιεχόμενο του αρχείου `FIRESTORE_RULES.txt`
3. Πάτα **Publish**

Αυτό αντικαθιστά το test mode (που λήγει σε 30 μέρες) με μόνιμους κανόνες. Δεν θα ξαναχρειαστεί.

---

### Βήμα 2 — Ανέβασε τον κώδικα στο GitHub

**Χωρίς terminal — μέσω της ιστοσελίδας του GitHub:**

1. Πήγαινε **github.com** → φτιάξε λογαριασμό (αν δεν έχεις)
2. Πάτα το **+** πάνω δεξιά → **New repository**
3. Όνομα: `4crete` → **Create repository**
4. Στη σελίδα που ανοίγει → **uploading an existing file**
5. Σύρε ΟΛΑ τα αρχεία αυτού του φακέλου μέσα (package.json, vite.config.js, index.html, ο φάκελος src/, κλπ.)
6. Πάτα **Commit changes**

---

### Βήμα 3 — Σύνδεσε με Vercel (αυτόματο deploy)

1. Πήγαινε **vercel.com** → **Sign up** → επίλεξε **Continue with GitHub**
2. **Add New → Project**
3. Επίλεξε το `4crete` repository → **Import**
4. Άφησε τις προεπιλογές (Vercel αναγνωρίζει αυτόματα ότι είναι Vite)
5. Πάτα **Deploy**

Μετά από ~1 λεπτό παίρνεις ένα μόνιμο URL, π.χ. `4crete.vercel.app`

**Τέλος!** Το URL δουλεύει σε όλες τις συσκευές, για πάντα, δωρεάν.

## 📱 Πώς χρησιμοποιείται στο τουρνουά

| Συσκευή | Τι κάνει |
|---------|----------|
| **Laptop 1** | Admin (PIN: 4444) → Live → Διαχείριση Γήπεδο 1 → TV 1 με HDMI |
| **Laptop 2** | Admin (PIN: 4444) → Live → Διαχείριση Γήπεδο 2 → TV 2 με HDMI |
| **Κινητά θεατών** | Ανοίγουν το URL — βλέπουν τα πάντα live (χωρίς PIN) |
| **Display tab** | Πάτα Fullscreen για καθαρό scoreboard στην TV |

Όλα συγχρονίζονται αυτόματα: σκορ, χρόνος, βαθμολογία, bracket, MVP, στατιστικά.

---

## 🔑 Admin PIN

Ο κωδικός διαχείρισης είναι **4444**. Για να τον αλλάξεις, βρες τη γραμμή `const ADMIN_PIN = "4444";` στο `src/App.jsx`.
